# 💻 Curso de Desenvolvimento Web

Este repositório contém todas as anotações, exercícios e projetos práticos desenvolvidos durante o meu aprendizado em Desenvolvimento Web. O objetivo é consolidar o conhecimento e servir como um portfólio de evolução técnica.

## 🚀 Tecnologias em Estudo
Abaixo estão as principais ferramentas e linguagens abordadas no curso:

*   **Fundamentos:** HTML5, CSS3 e JavaScript (ES6+)
*   **Frontend:** React.js / Vue.js / Angular
*   **Estilização:** Tailwind CSS / Sass / Bootstrap
*   **Backend:** Node.js / Express
*   **Banco de Dados:** MongoDB / PostgreSQL / MySQL
*   **Ferramentas:** Git, GitHub, VS Code e Docker

## 📂 Estrutura do Repositório
O repositório está organizado por módulos/seções para facilitar a navegação:

*   `/modulo-01-fundamentos`: HTML, CSS e lógica de programação.
*   `/modulo-02-frontend`: Componentização, estados e APIs com React.
*   `/modulo-03-backend`: Criação de APIs RESTful e autenticação.
*   `/projetos`: Aplicações completas desenvolvidas durante o curso.
